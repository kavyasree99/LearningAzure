### Now we have to create VM

We need to fill manditorty fields such as
**Project details:**
1. Select subcription -> Resouce Group
**Instance Details:**
2. Provide VM Name
3. Select a region
4. Select Security type
5. **Image:** (According to your project requirement)
6. **Size**
7. Authentication type (SSH/UserName,Password)
**Networking:** There are three main fields are need to updated
We Can create VNet Before creating VM or while creating
8. Virtual Network name
9. Subnet
10. Public IP

These are the Manditory fields to create VM and this help us in Automating VM creation Process
